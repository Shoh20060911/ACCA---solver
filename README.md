# ACCA AI Problem Solver

An AI-powered web app that solves any ACCA exam question instantly with full step-by-step working.

## Features
- Supports all ACCA papers: FA/F3, MA/F2, TX/F6, FR/F7, AA/F8, FM/F9, SBR, AFM
- Step-by-step worked solutions with final answer highlighted
- Built-in example questions for every paper
- Clean, mobile-friendly interface
- Powered by Google Gemini AI (free)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- AI: Google Gemini 2.0 Flash API
